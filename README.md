# Endless Bullseye

Esqueleto inicial para un juego en Godot 4.5.1.

## Estructura
- `project.godot`: configuración del proyecto.
- `scenes/main.tscn`: escena principal.
- `scripts/main.gd`: lógica inicial.
- `assets/bulleye.png`: textura de la diana incluida en el repositorio y usada en la interfaz.

## Controles
- `Enter` / `Space`: suma puntos para validar entrada.
